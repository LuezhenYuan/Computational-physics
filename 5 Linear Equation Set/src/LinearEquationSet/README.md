# 高斯消去法
1. 对于线性方程组Ax=B, 输入系数矩阵A, 值向量B. 初始化x矩阵
2. 消元的数组更新顺序:

```
B(k) ...> B(i)
A(k,j) ...> A(i,j)
```

```
for k=1:(n-1)
	ColumnWatch(&A);//可以应用列主元素消去法
	B(k) =B(k)/A(k,k)
	for (j=k+1):n
		A(k,j)=A(k,j)/A(k,k)
		B(j)=B(j)-A(j,k)*B(k);
		for (i=k+1):n
			A(i,j)=A(i,j)-A(i,k)*A(k,j);

B(n)=B(n)/A(n,n)
```

3. 求方程组的解

```
for(i=n;i>=1;i--)
	x[i] = B[i];
	for(j=i+1;j<=n;j++)
		x[i]-=A(i,j)*x[j];
```

# 改进的列主元素消去法
原始的列主元素消去法需要将系数矩阵A的行互换(如第k和第j行).这通常需要申请一片暂时的空间,进行3*(n-k+1)次赋值操作.

改进的方法是将第j行加到第k行上去(A(k,k)同号时加,异号时减).方程组的解不变.
